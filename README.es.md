Habilidad de Auto Orquestador para OpenClaw.

## Qué Soluciona
Orquestador potenciado por IA para tareas de datos.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque de seguridad primero
- Soporte para reversión

## Uso
/auto-orchestrator

## Ejemplo
```
Analizar código para detectar problemas de datos.
```

## Instalación
Esta habilidad está automáticamente disponible en OpenClaw.
